# bioKitPy

**bioKitPy** is a Python module I developed to enhance my programming skills while exploring bioinformatics concepts. It provides basic tools for handling and analyzing genomic sequences, with an independent implementation inspired by BioPython.

---

## Features

- Manage genomic sequences (DNA, RNA, Proteins).
- Convert DNA to RNA and RNA to proteins.
- Validate nucleotide sequences.
- Translate codons into amino acids.
- Perform common sequence operations.
